### Creating a Node Project

Create a project/app folder
> mkdir app-name
> cd app-name   
> create a root file(index.js)
> npm init
> npm install package-name (npm install date-fns)
> create the .gitignore file and add node_modules
> npm i date-fns
> npm i uuid
